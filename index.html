  • 2. Controls
    None: Nothing is required.
    Low: Activity logging, direct connection (connect button).
    Ticket Only: ITSM ticket.
    Normal: Activity logging, direct connection (connect button), ITSM ticket.
    For critical accounts:
      Business Approval: Activity logging, direct connection (connect button), approvals.
      Approval: Approval, ITSM Ticket.
      High: Activity logging, direct connection (connect button), approvals, ITSM ticket.
      High Risk: Video recording/Keystroke logging, activity logging, direct connection (connect button).
      High Risk Business Approval: Video recording/Keystroke logging, activity logging, direct connection (connect button), approvals.
    Info If you are not sure about which control to choose yet, please click here for more info

    Controls Notice

    Production Environments usually require Normal or High as the level of control.

    This enforces validation of either a TFS item number or an ITSM item number need to be referenced in order to proceed with account checkout.

    For Non-Prod accounts there is no need for such strict controls. 'Low' and 'None' are perfectly acceptable.

    Nonetheless: please consult with your ITRM advisor in order to assess the correct controls for your privileged accounts.

    Disabled: Enforces connect button usage. Show and Copy are disabled.
  • 4. Password Management
    Auto: Password is managed by ITPA. Password is reset after each use and periodically.
    External: Password is not managed by ITPA. Password is not reset after each use and multiple individuals can check out an account at one time (Multi-use and No Exclusivity).
    Manual: Password is managed by ITPA. Password is not reset after each use and multiple individuals can check out an account at one time (Multi-use and No Exclusivity).
    ManualOnUse: Password is not managed by ITPA. Password must be reset after each use. Users must reset the password in the target system and set the password in ITPA to match the password in the target system.
    On Use: Password is managed by ITPA. Password is reset after each use.
    Periodic Only: ITPA rotates this password periodically.
